# In main css file:

## Customizing breakpoint
```html
@theme {
  --breakpoint-xs: 30rem;
  --breakpoint-2xl: 100rem;
  --breakpoint-3xl: 120rem;
}
```

## Customizing color class values
```html
@theme {
  --color-primary: #2563eb;     /* blue-600 */
  --color-primary-light: #3b82f6; 
  --color-primary-dark: #1e40af;

  --color-secondary: #9333ea;   /* purple-600 */
  --color-secondary-light: #a855f7;
  --color-secondary-dark: #7e22ce;
}
```

### using this way @theme{}, we can customize any class and also add new custom class
